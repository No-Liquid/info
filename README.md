# NoLiquid Landing Page

A clean, modern landing page for NoLiquid - a decentralized trading platform with zero liquidation risk.

## Design

- **Theme**: White background with black font
- **Typography**: JetBrains Mono, JetBrains Mono Fallback
- **Style**: Minimalist, professional, and modern

## Features

- Responsive design
- Clean navigation
- Hero section with CTA
- Features showcase
- How it works section
- Stats display
- Footer with links

## Usage

Simply open `index.html` in your web browser to view the landing page.

## Customization

All styles are contained in `styles.css` and can be easily modified to fit your needs.
